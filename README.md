### Gem:

`````
gem 'devise'
`````

* Bootstrap framework gem
`````

`````
gem 'screenshot_machine'
`````

* Gem to search the *Title* in the URL
`````
gem 'nokogiri'
`````

* Bootstrap for *form_for*
`````
gem 'bootstrap_form'
